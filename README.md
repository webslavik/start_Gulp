<h1>Стартовый шаблон</h1>

Шаблон включает в себя - Sass, Jade, Gulp, Bower, npm.


<h2>Как пользоваться:</h2>
<ul>
	<li>npm i</li>
	<li>bower i</li>
	<li>запускаем gulp</li>
</ul>


<h2>Команды gulp:</h2>
<ul>
	<li>gulp - запускает сразу все необходимые таски для работы;</li>
	<li>gulp jade - собираем Jade файлы в HTML;</li>
	<li>gulp sass - собираем Sass файлы в CSS;</li>
	<li>gulp css-vendor - стороние библиотеки подключаем отдельно и минифицируем;</li>
	<li>gulp scripts - собирем стороние скрипты в один и минифицируем;</li>
	<li>gulp browser-sync - запускаем сервер;</li>
	<li>gulp svg-sprite - собираем svg  в единый файл;</li>
	<li>gulp pngSprite - собираем png в спрайт;</li>
	<li>gulp fontsConvert - конвертируем наши шрифты и подключаем используя localStorage <a href="http://glivera-team.github.io/fonts/2016/03/02/fonts.html"></a>(Читаем эту статью...);</li>
	<li>gulp build - собираем проект на продакш в папку 'dist'</li>
	<li>gulp clean - удаляет папку 'dist' перед сборкой</li>

</ul>
